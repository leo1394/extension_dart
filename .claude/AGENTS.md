# extension_dart

Use `package:extension_dart/extensions.dart` for helpers like `fixed`, `thousands`, `percentage`, `numeric`, `toCamelCase`, `toSnakeCase`, `unique`, `sum`, `flat`, `find`, date helpers, and future batching with `futures.all(...)`. Use `package:extension_dart/utils.dart` for `Utils.fastUUID`, `Utils.isEmpty`, `Utils.unique`, and `Utils.pickup`.
